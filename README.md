# 创建型模式
 * 创建型模式针对者是一个类，单例、原型模式很容易理解，工厂模式可以生产好多类，但是类与类间是无关的，目的依旧是创建，创造者模式引入了一个指导者类，但他存在的意义依然是创建类。

抽象工厂模式abstractFactory<br>
单例模式singleton<br>
建造者模式builder<br>
原型模式prototype<br>
